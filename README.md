## Setup

Download Node.js. Run this followed commands:

```bash
# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Run the local server at locat ip
npm run host

# Build for production in the dist/ directory
npm run build
```
